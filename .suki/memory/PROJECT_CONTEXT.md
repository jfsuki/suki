# Project Context
Este archivo contiene la visión técnica y arquitectura del proyecto SUKI.

## Visión General
SUKI es un AI Application Operating System (AI-AOS) diseñado para ejecutarse en entornos PHP 8.1+ (cPanel).

## Arquitectura
- **Kernel**: Procesamiento de lenguajes y enrutamiento de intenciones.
- **Tools**: Habilidades atómicas registradas en el SkillRegistry.
- **Memory**: Capa de persistencia jerárquica (Largo Plazo / .suki).

## Stack Tecnológico Relevante
- PHP 8.1+
- SQLite (Metadatos y Registro)
- Filesystem (Memoria y Contratos)

# Validation Engine — EN
## Goals
- Declarative rules
- Same rules for UI + backend validation
- Clear error model

## Rule types
- required, min, max
- regex
- type checks
- cross-field rules
- formula validation (safe eval)

## Error model
{ field, rule, message, severity, context }

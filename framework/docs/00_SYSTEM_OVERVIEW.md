# System Overview — EN
## Mission
Build a “mother app” platform that generates many customized business apps without code changes, using JSON contracts.

## What users see
Simple UI. Non-technical entrepreneurs can operate it. Accounting/DIAN complexity is hidden.

## What the platform guarantees
- Robust engine
- Security & multi-tenant
- Audit trail
- DIAN-ready architecture
- Backward-compatible contracts

## Core outcomes
- Faster delivery (templates + configs)
- Fewer regressions (No-Break policy + versioning)
- Scalable monetization (modules, vertical packs)
- Dev assistant that generates contracts, views, and DB scaffolding

# Visión General del Sistema — ES
## Misión
Construir una plataforma “app madre” que genere múltiples apps de negocio personalizadas sin tocar código, usando contratos JSON.

## Lo que ve el usuario
Interfaz simple. Emprendedor no técnico puede usarlo. La complejidad contable/DIAN queda oculta.

## Lo que garantiza la plataforma
- Motor robusto
- Seguridad y multi-tenant
- Auditoría
- Arquitectura lista para DIAN
- Contratos retrocompatibles

## Resultados
- Entrega rápida (plantillas + configs)
- Menos regresiones (No-Break + versionado)
- Monetización escalable (módulos y paquetes por sector)
- Asistente dev que genera contratos, vistas y base de datos
